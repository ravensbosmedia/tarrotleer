# Lenormand-kaartafbeeldingen

Plaats hier de **36 kaartafbeeldingen**. Het platform pakt ze automatisch op via
het pad `/lenormand/NN.jpg` (zie `src/data/lenormand/kaarten.ts`).

## Regels
- Bestandsnaam = **kaartnummer met twee cijfers**, extensie **`.jpg`**: `01.jpg` … `36.jpg`.
- Staat een bestand er (nog) niet, dan toont de kaart automatisch het emoji-symbool — niets breekt.
- Aanbevolen: staand (portret), gelijke verhouding, bijv. ~600×1000 px.
- Wil je een ander formaat dan `.jpg`? Pas dan het `afbeelding`-veld in `kaarten.ts` aan.
