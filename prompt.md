# Nederlandse Tarot Leggingen Web Applicatie

## Project Beschrijving
Een volledig functionele tarot applicatie in het Nederlands, gebouwd met React, TypeScript en Firebase. De applicatie biedt verschillende tarot leggingen aan, van een enkele dagkaart tot het complexe Keltisch Kruis, met AI-gestuurde interpretaties en een interactieve chat functionaliteit.

## Technische Stack
- React met TypeScript
- Firebase voor database en authenticatie
- Tailwind CSS voor styling
- Lucide React voor iconen
- OpenAI API voor interpretaties en chat

## Kernfunctionaliteiten

### 1. Leggingen
- Dagkaart (1 kaart)
- 3 Kaarten Legging (verleden, heden, toekomst)
- Keltisch Kruis (10 kaarten)
- Relatie Legging (5 kaarten)
- Carrière Legging (4 kaarten)

### 2. Kaart Selectie
- Handmatige kaartselectie
- Automatisch schudden
- Visuele weergave van geselecteerde kaarten
- Optionele vraagstelling voor leggingen

### 3. Interpretaties
- Basis interpretatie van kaarten
- AI-gegenereerde persoonlijke interpretaties
- Interactieve chat voor vragen over de legging
- Gedetailleerde kaartbetekenissen

### 4. Educatieve Content
- Tarot basis informatie
- Geschiedenis van tarot
- Interpretatie handleiding
- Beginner's gids

### 5. Extra Functionaliteiten
- PDF export van leggingen
- Print functionaliteit
- Responsief ontwerp
- Nederlandse interface

### 6. Kaart Database
- 78 kaarten met Nederlandse en Engelse namen
- Uitgebreide beschrijvingen
- Rechtopstaande en omgekeerde betekenissen
- Speciale interpretaties voor liefdesleggingen

### 7. Admin Functionaliteit
- Kaart beheer systeem
- Document upload systeem
- Importeer functionaliteit
